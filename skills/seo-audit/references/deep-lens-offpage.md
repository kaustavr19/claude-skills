# Deep Lens — Off-page & Authority

Run only when selected. Everything that builds a site's authority and trust from
*outside* the site. Note up front: deep backlink data needs a dedicated tool
(Ahrefs/Semrush/Majestic) or Search Console; without it, assess qualitatively
and state the limit.

## Checks
- **Backlink profile (if data available):** referring-domain count and quality,
  anchor-text distribution, dofollow/nofollow mix, toxic/spam links. Quality and
  topical relevance beat raw volume.
- **E-E-A-T signals** (Experience, Expertise, Authoritativeness, Trust):
  named authors with bios and credentials, `Author`/`Person` schema, about/
  contact transparency, editorial standards, citations to reputable sources.
  Especially critical for YMYL (your-money-your-life) topics.
- **Brand signals:** branded search demand, consistent NAP, presence on
  Wikipedia/Wikidata, social profiles linked via `sameAs`.
- **Digital PR / link-earning angles:** identify genuinely linkable assets the
  site has or could create (original data, tools, definitive guides) — more
  durable than link building.
- **Unlinked mentions:** brand mentioned without a link → outreach opportunity.
- **Competitor gap (if tooling allows):** referring domains competitors have
  that this site doesn't.

## Reporting
Be explicit about data limits when you lack a backlink tool. Favour durable,
white-hat recommendations (earn links via assets and PR) over volume tactics.
E-E-A-T improvements are often the highest-leverage, fully on-site lever.
