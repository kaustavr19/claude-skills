# Example Skill

## When to use this skill

Describe the situations that should trigger the skill. Be specific — the
`description` field in the frontmatter is what Claude matches against, so make it
rich with trigger phrases and scenarios.

## Instructions

1. Step-by-step guidance for what Claude should do when this skill runs.
2. Keep it focused on one job done well.
3. Reference any supporting files in this folder by relative path.

## Supporting files (optional)

You can include reference docs, scripts, or templates alongside `SKILL.md`:

```

├── SKILL.md          # required — the entry point
├── reference.md      # optional — extra context Claude can read on demand
└── templates/        # optional — files the skill produces or uses
```
